# 笔记库

一个多层嵌套的知识库示例，用于展示 **MD Go 侧边栏的目录树**能力——展开/折叠、嵌套层级、全部展开/折叠、文件名过滤。

## 目录结构

```
笔记库/
├── 技术/
│   ├── 前端/
│   │   ├── React要点.md
│   │   └── CSS布局.md
│   ├── 后端/
│   │   ├── Go并发编程.md
│   │   └── 数据库设计.md
│   └── 工具/
│       └── Git速查.md
├── 生活/
│   ├── 阅读/
│   │   └── 2026书单.md
│   └── 旅行/
│       └── 想去的地方.md
└── 归档/
    └── 历史笔记.md
```

共 **3 层嵌套**，覆盖技术、生活、归档三大类。

## 在 MD Go 中体验

1. 「打开文件夹」选中 `examples/` 目录
2. 侧边栏会显示完整的目录树
3. 试试这些操作：
   - 点击文件夹的 `▶` / `▼` 展开 / 折叠
   - 侧边栏顶部的「全部展开」「全部折叠」
   - 搜索框输入文件名快速过滤
   - 右键文件 / 文件夹：新建、重命名、删除（走回收站）

> 目录树是 MD Go「工作区」体验的核心——把散落的 Markdown 文件组织成结构化的知识库。

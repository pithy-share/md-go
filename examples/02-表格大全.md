# 表格大全

MD Go 支持完整的表格编辑：工具栏「插入表格」生成 3×3 表格，「表格菜单」增删行列。

## 基础表格

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 新建文档 | Ctrl+N | 创建空白标签页 |
| 打开文件 | Ctrl+O | 选择本地 Markdown 文件 |
| 保存 | Ctrl+S | 写入磁盘 |
| 另存为 | Ctrl+Shift+S | 保存为新文件 |
| 查找 | Ctrl+F | 当前文档内搜索 |
| 工作区搜索 | Ctrl+Shift+F | 跨文件全文搜索 |
| 命令面板 | Ctrl+P | 搜索并执行命令 |

## 对齐方式

在分隔行用 `:` 控制列对齐：

| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 靠左 | 居中 | 靠右 |
| Markdown | 表格 | 语法 |
| 2026 | 06 | 14 |

## 含格式的单元格

单元格内可包含 **粗体**、*斜体*、`代码`、[链接](https://github.com) 等格式：

| 类型 | 示例 | 状态 |
|------|------|:----:|
| 编程语言 | `Go`、`TypeScript`、`Python` | ✅ |
| 框架 | **Wails**、*React* | ✅ |
| 仓库 | [md-go](https://github.com/pithy-share/md-go) | 🎉 |
| 协议 | MIT | 📄 |

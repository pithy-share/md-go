# MD Go 示例文档

这是一组用于展示 [MD Go](../README.md) 编辑能力的 Markdown 文档，也可作为语法速查和写作模板。

## 文档索引

| 文件 | 展示内容 |
|------|---------|
| [01-语法速查.md](./01-语法速查.md) | 标题、强调、列表、引用、链接、分割线 |
| [02-表格大全.md](./02-表格大全.md) | 各种对齐方式的表格 |
| [03-代码合集.md](./03-代码合集.md) | 多语言代码块语法高亮 |
| [04-笔记模板.md](./04-笔记模板.md) | 任务列表、会议记录、读书笔记 |

## 推荐玩法

1. 在 MD Go 里用「打开文件夹」选中本 `examples/` 目录，作为工作区打开
2. 侧边栏会出现文件树，可点击切换
3. 文档里的 `[本地链接](./*.md)` 可直接跳转
4. 试试这些快捷键：
   - `Ctrl+P` —— 命令面板
   - `Ctrl+Shift+F` —— 跨文件全文搜索
   - `Alt + ←` / `Alt + →` —— 浏览历史后退 / 前进
   - `Ctrl+Shift+E` / `Ctrl+Shift+P` —— 导出 HTML / PDF

# Markdown 语法速查

本文件汇集 MD Go 支持的基础 Markdown 语法，方便随时查阅。在所见即所得模式下，这些语法会实时渲染。

## 标题

用 `#` 的数量表示标题层级，共六级：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

## 文本强调

- **粗体** 文本
- *斜体* 文本
- ***粗斜体***
- ~~删除线~~
- `行内代码`
- [超链接](https://wails.io)

也可以**混合使用 *多种* `格式`** 在同一句话里。

## 列表

### 无序列表

- 第一项
- 第二项
  - 嵌套子项 A
  - 嵌套子项 B
    - 更深层级
- 第三项

### 有序列表

1. 步骤一：打开文件夹
2. 步骤二：选择 Markdown 文件
3. 步骤三：开始编辑

## 引用

> 这是一段引用文本。
>
> 引用可以跨多个段落。
>
> > 引用还能嵌套。
>
> 引用内部也可以包含 **强调**、`代码` 和 [链接](https://github.com)。

## 分割线

用三个及以上的 `-` 或 `*` 生成分割线：

---

上方与下方由分割线隔开。

## 转义字符

用反斜杠转义 Markdown 特殊字符：

- `\*这不是斜体\*`
- `\#这不是标题`
- `\`反引号可用反斜杠转义\``
